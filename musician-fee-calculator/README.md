  # Musician Fee Calculator ♫

I developed this simple Python script to help me plan and standardize my gig quotes. It’s a tool for my personal workflow as a singer, ensuring that I don't overlook basic costs when sending proposals to clients.
</div> <br>

## ✧ Purpose
The goal is to provide a generic and organized way to compose budget values based on:
- **Travel distance** (fuel and vehicle wear).
- **Audience size** and **Show duration**.
- **Equipment and Service fees**.

## ✧ How it works
The script calculates a final estimate by summing a base fee with variable costs, helping me maintain a consistent pricing strategy for every performance.

## ✧ How to run
1. Ensure you have Python installed.
2. Run the script via terminal:
   ```bash
   musician-fee-calculator.py

# Desafio DIO: Sistema Bancário - 🐍Python

Este projeto faz parte do bootcamp da DIO e consiste na criação de um sistema bancário simples em Python. O objetivo principal foi aplicar os conceitos básicos de lógica de programação, estruturas de repetição (`while`) e condições (`if/elif/else`).

## Funcionalidades

O sistema simula as operações básicas de uma conta bancária através de um menu interativo no terminal:

* **[d] Depósito:** Permite adicionar valores positivos à conta. Todos os depósitos são armazenados no histórico do extrato.
* **[s] Saque:** Permite levantar dinheiro da conta, respeitando as seguintes regras:
    * Limite máximo de **3 saques** diários.
    * Limite máximo de **R$ 500.00** por saque.
    * Não é permitido sacar se não houver saldo suficiente.
* **[e] Extrato:** Exibe a listagem de todas as movimentações realizadas (depósitos e saques) e o saldo atual da conta. Se não houver movimentações, exibe uma mensagem informativa.
* **[q] Sair:** Encerra o programa.

## Tecnologias Utilizadas

* **Linguagem Python**
* **VS CODE**
* Conceitos aplicados: Variáveis, Strings formatadas (f-strings), Métodos de string (`.strip()`, `.lower()`), Estruturas de Decisão e Laços de Repetição.

# Challenge: Simple Banking System in Python

This project is part of the DIO bootcamp and consists of creating a simple banking system using Python. The main objective was to apply basic programming logic concepts, repetition loops (`while`), and conditional structures (`if/elif/else`).

## Features

The system simulates the basic operations of a bank account through an interactive terminal menu:

* **[d] Deposit:** Allows adding positive amounts to the account. All deposits are stored in the statement history.
* **[s] Withdrawal:** Allows withdrawing money from the account, respecting the following rules:
    * Maximum limit of **3 withdrawals** per day.
    * Maximum limit of **R$ 500.00** per withdrawal.
    * It is not allowed to withdraw if there is insufficient balance.
* **[e] Statement:** Displays a list of all transactions made (deposits and withdrawals) and the current account balance. If no transactions have occurred, it displays an informative message.
* **[q] Quit:** Exits the program.

## Technologies Used

* **Python Language**
* **VS Code**
* Applied concepts: Variables, Formatted strings (f-strings), String methods (`.strip()`, `.lower()`), Decision Structures, and Loops.
